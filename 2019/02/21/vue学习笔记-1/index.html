props:
父子组件间通信的基本方式
属性值的 2 大类型:
一般: 父组件–>子组件
函数: 子组件–>父组件
隔层组件间传递: 必须逐层传递(麻烦)
兄弟组件间: 必须借助父组件(麻烦)
vue 自定义事件
子组件与父组件的通信方式
用来取代 function props
不适合隔层组件和兄弟组件间的通信
pubsub 第三方库(消息订阅与发布)
适合于任何关系的组件间通信
slot
通信是带数据的标签
注意: 标签是在父组件中解析
vuex
多组件共享状态(数据的管理)
组件间的关系也没有限制
功能比 pubsub 强大, 更适用于 vue 项目
